# EDA Project

This repository contains exploratory data analysis (EDA) work using Python libraries such as **NumPy** and **Pandas**. The focus is on learning and applying fundamental data manipulation techniques.

---

## Contents

- **Numpy/**  
  Examples and practice files demonstrating NumPy operations.  


- **Pandas/**  
  Work on Pandas including the Big Basket dataset and exercises on filtering, sorting, renaming, and handling missing values.  


- **.DS_Store**  
  System file generated by macOS.  
  *Last updated: 3 minutes ago*

- **README.md**  
  Documentation for the repository.  

---

## Key Topics Covered

1. **NumPy Basics**  
   - Array creation and manipulation  
   - Mathematical operations  
   - Indexing and slicing  

2. **Pandas Basics**  
   - Selecting rows and columns  
   - Filtering with conditions  
   - Sorting with `.sort_values()`  
   - Renaming columns with `.rename()`  
   - Checking and handling missing values with `.isnull()`, `.dropna()`, and `.fillna()`  

3. **Big Basket Dataset Analysis**  
   - Practical application of Pandas methods  
   - Data cleaning and preparation  
   - Exploratory insights
